# Object hooks

{: .fs-6 .fw-300 }

## What is a hook?

A hook can be any executable file

## Postcreate hooks

### Create a hook folder

    mkdir [OBJ DIR]/postcreate

### Put your hook scripts inside

The hooks are executed in alphabetical order

    $ ls -1 [OBJ DIR]/postcreate

    000_hook.sh
    001_hook.sh
